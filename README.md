# Description

This script that shows the players weapons they are carrying on there body.
(Handguns and Small Guns on Hip, Rifles and Machine GUns on Back)
# Preview
Photos In Discord!
# Features

- Shows Pistols and Small Weapons on Hip
- Shows Rifles and Larger Guns/Weapons On Back
- The script is configurable.
- Able to add more guns.
- Works on OX and QB Inventory
- Removes gun of body when drawn.
- Syncronized across network

# Expanding on the config
Make sure to check your weapons.lua file to find weapon spawn names if you wish to add more.

# Config
System Comes Default With:
- All Standard Weapons

- "handgun" To Put Weapon on Hip
- "rifle" To Put Weapon on Back

Positions are Pre Configured.
